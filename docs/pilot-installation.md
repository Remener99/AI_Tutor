# Семпейс AI pilot installation

Эта инструкция предназначена для пилотной установки расширения без Chrome Web Store.

## Что внутри pilot-package

- `extension/` - распакованное расширение для установки через Developer Mode.
- `chrome-mv3-prod.zip` - ZIP-архив сборки расширения.
- `privacy-policy.md` - privacy policy.
- `data-disclosure.md` - пояснение, какие данные читает расширение.
- `test-instructions.md` - сценарии проверки.

## Установка в Google Chrome

1. Распакуйте архив `sempace-ai-pilot-0.1.5.zip`.
2. Откройте Chrome.
3. Перейдите на `chrome://extensions/`.
4. Включите **Режим разработчика**.
5. Нажмите **Загрузить распакованное расширение**.
6. Выберите папку `extension`.
7. Откройте страницу Synergy LMS.
8. Проверьте, что боковая панель `Семпейс AI` открывается и видит страницу LMS.

## Установка в Яндекс Браузер

1. Распакуйте архив `sempace-ai-pilot-0.1.5.zip`.
2. Откройте Яндекс Браузер.
3. Перейдите на `browser://extensions/`.
4. Включите **Режим разработчика**.
5. Нажмите **Загрузить распакованное расширение**.
6. Выберите папку `extension`.
7. Откройте страницу Synergy LMS.
8. Проверьте, что боковая панель `Семпейс AI` открывается и видит страницу LMS.

Если `browser://extensions/` не открывается, используйте `chrome://extensions/`.

## Что расширение читает

Расширение работает только на поддерживаемых страницах Synergy LMS и читает:

- URL и заголовок текущей LMS-страницы;
- названия дисциплин и тем;
- видимый прогресс обучения;
- сроки семестра;
- текст PDF, который пользователь сам загружает в расширение;
- вопросы и ответы, которые пользователь вводит в AI-тьютор.

Расширение не читает пароли, платежные данные, личные сообщения и историю браузера вне поддерживаемых LMS-страниц.

## Важное ограничение

На страницах итоговых тестов и аттестаций AI-функции блокируются. Расширение предназначено для изучения материалов, а не для прохождения тестов за студента.

## Быстрая проверка после установки

1. Открыть Synergy LMS.
2. Убедиться, что расширение видит страницу LMS.
3. Сформировать маршрут недели.
4. Загрузить PDF занятия.
5. Проверить AI-чат по PDF.
6. Проверить быстрые функции: мини-конспект, глоссарий, практическая работа.
7. Открыть страницу итогового теста и убедиться, что AI-функции заблокированы.
