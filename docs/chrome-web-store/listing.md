# Chrome Web Store listing

## Product name

Семейс AI

## Short description

AI-тьютор для Synergy LMS: маршрут недели, разбор PDF, мини-конспект, глоссарий и практические задания.

## Detailed description

Семейс AI помогает студенту работать с учебными материалами в Synergy LMS.

Расширение открывается в боковой панели Chrome и формирует персональный маршрут недели на основе доступных занятий, прогресса и сроков семестра. Пользователь может загрузить PDF занятия и получить объяснение материала, мини-конспект, глоссарий, практическую работу и AI feedback.

Основные возможности:

- маршрут недели по текущим дисциплинам и темам;
- AI-разбор PDF лекции;
- чат с AI-тьютором по загруженному материалу;
- мини-конспект, глоссарий и практическая работа;
- AI feedback по ответам студента;
- блокировка AI-функций на страницах итоговых тестов.

Расширение не проходит тесты за студента, не подставляет ответы в LMS и не изменяет данные на странице. Оно работает как помощник для самостоятельного изучения материалов.
