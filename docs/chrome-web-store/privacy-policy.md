# Privacy Policy for Семпейс AI

Last updated: 2026-06-10

Семпейс AI is a Chrome extension that helps students study materials in Synergy LMS. The extension opens as a sidebar, reads the current LMS page, builds a weekly study route, and sends selected study context to the backend AI service when the user requests AI features.

## What data the extension reads

The extension can read the following information from Synergy LMS pages:

- current page URL and page title;
- discipline and topic names;
- visible learning progress and completion status;
- semester dates and study route context;
- text extracted from PDF files uploaded by the user inside the extension;
- user questions and answers typed into the extension.

The extension does not intentionally collect passwords, payment data, personal messages, browser history outside supported LMS pages, or answers from final test pages.

## How data is used

The data is used only to provide the extension features:

- generate a weekly learning route;
- explain uploaded PDF materials;
- generate a mini-summary, glossary and practice task;
- provide AI feedback for learning;
- block AI assistance on final test pages.

## Data transfer

When the user uses AI features, the extension sends the required study context to the Семпейс AI backend. The backend uses an OpenAI-compatible AI provider to generate responses.

## Data storage

The extension stores local settings and temporary study state in Chrome extension storage. Uploaded PDF text is converted into a temporary material context for AI tutoring. The product is designed for short-lived educational context, not long-term storage of personal files.

## Data sharing

We do not sell user data. We do not use user data for advertising. We do not transfer user data to third parties except the backend infrastructure and AI provider required to process user requests.

## Final tests and exams

The extension detects final test and assessment pages and blocks AI tutoring features there. Семпейс AI is intended for learning support, not for completing exams or tests for the user.

## Contact

For privacy questions, contact the developer using the support email listed in the Chrome Web Store item.
